A fun, dinosaur-themed calculator built with HTML, SCSS, and JavaScript.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- Responsive design for mobile, tablet, and desktop
- Keyboard support
- Error handling for invalid operations

## How to Use

### Clicking

- Click number buttons to enter numbers
- Click operator buttons (+, -, \*, /) to choose an operation
- Click = to see the result
- Click C to clear and start over

### Keyboard

- Type numbers (0-9) to enter them
- Type operators (+, -, \*, /) to choose an operation
- Press Enter to calculate
- Press Backspace or Escape to clear

## Files

- `calculator.html` - Main HTML structure
- `scss/calculator-scss/calculator.css` - Compiled styles
- `javascript/calculator.js` - Calculator logic
